# Logfile_Manager 

면접과제 : C++ 표준 라이브러리를 활용한 로그 기록 및 관리하는 시스템
<br><br>
 
Tool: Visual Studio 2022
<br><br>

주요기능
openLogFile메소드 입력값 파일이 없으면 만들고 map에 등록
writeLog메소드	  파일에 타임스탬프와 함께 로그 기록
readLogs메소드	  파일의 모든 내용을 읽어 반환
closeLogFile메소드 맵에서 해당 파일을 제거하고 리소스 해제
